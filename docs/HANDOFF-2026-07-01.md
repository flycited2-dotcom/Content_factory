## ЗАЧЕМ (ключевой момент)
Владелец продаёт климат/бытовую технику. Нужен **автоматический контент-завод**: берём товары
из БД склада (oasis) → считаем цену → формируем **B2B-карточку** (заголовок + цена + кол-во +
«ключевые особенности»/УТП) → генерим премиальную **картинку** (фотоагент через ChatGPT) →
ревизия правилами → публикуем в **Telegram-канал для B2B-покупателей**. Канал **B2B**, поэтому
НЕ розничные призывы («подберём/поможем»), а **характеристики/УТП** — как в «боте остатков»
(stock_report_bot). Сегодня кондиционеры, дальше — бойлеры/холодильники/стиралки (категория-
независимо). Генерация картинок должна быть **отказоустойчивой** (десктоп владельца часто
обесточен → подключили ноутбук как второй генератор с авто-failover).

## ЧТО СДЕЛАНО (эта серия сессий)
### A. Фаза 1 контент-завода (M0–M6) — развёрнута
Полный конвейер: `bot/commands` (/plan, /approve…) + `orchestrator/{tasks,queue,scheduler,plans,

`cf-archive.timer` (ежечасный архив). `cf-cards.timer` — **disabled** (генерация идёт через агент).

### B. Авто-failover двух генераторов (в репо агента)
VPS-эндпоинт `/api/worker/lease` + таблица `workers`; агент перед взятием задачи спрашивает лиз,
неактивный → standby. Десктоп priority 1, ноут 2, порог ухода 15 мин, грациозный хендбэк. Ноут
настроен (`.env`: WORKER_ID=laptop, ключ `id_ritualb2b_claude` для туннеля), WatchDog в Планировщике
`RitualB2B_Watchdog_Laptop` (без всплывающих окон — `CREATE_NO_WINDOW`). **Десктоп ещё НЕ обновлён**
(был офлайн) — это Task 7.

### C. B2B-контент (одобрено владельцем)
Подпись = `Бренд Наименование — цена · N шт.` + разделитель + «Ключевые особенности»:
мощность (BTU·кВт·площадь из **реальных ТТХ**, не из битого btu_calc) + класс/инвертор/компрессор/
обогрев/шум/Wi-Fi + **✓-УТП**. ✓-УТП источники: **breeze** — Breez API (`ingest/breez.py`,
`/products/`, ~2655 поз., креды BREEZ_* в .env), **rusklimat** — поле «УТП», **daichi** — проза
«Описание». Умный дедуп (`content/specs.py::_dup_of_produced`). Те же rich-specs отдаются **агенту**
на генерацию (`cards_run` → `run_once(specs_fn=…)`) → картинка богатая, как у stockbot. Фильтр
наличия `stock>0`. `oasis_db.group_tech_rows max_specs 12→60`.

### D. Архив
`content/archive.py` + `archive_run.py` → `/opt/content-factory/archive/ГГГГ-ММ-ДД/` (ежечасно).
Google Drive не дорабатываем — агент уже льёт копии по режимам.
